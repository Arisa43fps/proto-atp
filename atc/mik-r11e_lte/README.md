# MikroTik R11e-LTE modem

Tested on a MiktoTik wAP R, RBwARP-2nD.

Follow installation guide on: https://openwrt.org/toh/mikrotik/common

Install following packages:
```
kmod-usb-acm
kmod-mii
kmod-usb-net
kmod-usb-net-rndis
chat
comgt
```
then install:
```
atc-mik-r11e_LTE
luci-proto-atc
```

Reboot

https://wiki.mikrotik.com/wiki/Manual:Netinstall
